**Status:** Done

## Objetivo
Declarar explicitamente no `classifier.py` quais constructs SAS são suportados (Tier 1/2) e garantir que qualquer construct fora do escopo seja classificado diretamente como Tier 3 (manual flag) sem tentativa de parsing — evitando silently wrong behavior.

## Contexto
A linguagem de macros SAS é Turing-complete. Constructs como `%SYSFUNC`, `CALL EXECUTE`, `%DO %WHILE`, `%MACRO` com geração dinâmica e recursão não são parseáveis deterministicamente. Tentar parsear esses casos produz resultados incorretos silenciosamente. O classifier precisa de um allowlist explícito de constructs suportados.

## Escopo
### Incluído
- Definir em `sas2dbx/analyze/classifier.py` um `SUPPORTED_CONSTRUCTS` dict com tier por construct:
  ```python
  SUPPORTED_CONSTRUCTS = {
      # Tier 1 — rule-based
      "DATA_STEP_SIMPLE": Tier.RULE,   # SET, WHERE, KEEP, DROP, RENAME
      "PROC_SQL": Tier.RULE,
      "PROC_SORT": Tier.RULE,
      "PROC_EXPORT": Tier.RULE,
      "PROC_IMPORT": Tier.RULE,
      "LIBNAME": Tier.RULE,
      # Tier 2 — LLM-assisted
      "DATA_STEP_COMPLEX": Tier.LLM,   # arrays, retain, by-group
      "PROC_MEANS": Tier.LLM,
      "PROC_SUMMARY": Tier.LLM,
      "PROC_FREQ": Tier.LLM,
      "MACRO_SIMPLE": Tier.LLM,        # %macro/%mend sem recursão/dinâmica
      # Tier 3 — manual flag (não parsear, preservar SAS original)
      "PROC_FORMAT": Tier.MANUAL,
      "PROC_REPORT": Tier.MANUAL,
      "PROC_TABULATE": Tier.MANUAL,
      "MACRO_DYNAMIC": Tier.MANUAL,    # CALL EXECUTE, %SYSFUNC, geração dinâmica
      "HASH_OBJECT": Tier.MANUAL,
      "UNKNOWN": Tier.MANUAL,          # default para qualquer construct não reconhecido
  }
  ```
- Implementar `classify_block(sas_block: str) -> ClassificationResult` que:
  1. Identifica o tipo de construct via regex/pattern matching
  2. Se não reconhecido → retorna `UNKNOWN` (Tier.MANUAL) sem tentar parsear
  3. Retorna `ClassificationResult(construct_type, tier, confidence)`
- Adicionar `confidence` score na classificação (1.0 para Tier 1, 0.8 para Tier 2, 0.0 para Tier 3)
- Testes com exemplos reais de cada categoria

### Excluído
- Implementação do transpiler em si — apenas classificação
- Parser completo de AST — o classifier usa pattern matching leve
