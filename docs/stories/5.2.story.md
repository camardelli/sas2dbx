# Story 5.2: Source Inventory Module — Extractor + Checker + Reporter + Gate

**Status:** Draft
**Sprint:** 5 — Inventory Foundation
**Depende de:** 5.1 (schemas.yaml list[dict])
**Bloqueada por:** 5.1

---

## Contexto

O fluxo atual transpila → preflight → descobre tabelas ausentes → deploy falha → fixer D3 tenta corrigir. O custo é alto: chamadas LLM desperdiçadas, deploy no Databricks, código destrutivo injetado. O Source Inventory Module roda ANTES do engine, verifica existência de todas as tabelas SOURCE no Databricks via UC REST API, e bloqueia a transpilação se tabelas estiverem ausentes.

## Objetivo

Criar `sas2dbx/inventory/` com 4 componentes: extractor (consome split_blocks()), checker (UC REST API sem warehouse), reporter (inventory_report.json + schemas.yaml tipado), gate (STRICT/LENIENT/FORCE/SKIP).

## Acceptance Criteria

- [ ] `extractor.py` consome `list[SASBlock]` — não parseia raw SAS
- [ ] Extractor classifica corretamente SOURCE / TARGET / INTERMEDIATE / UNRESOLVED
- [ ] LIBNAME desconhecido → role=UNRESOLVED, não SOURCE
- [ ] Classificação INTERMEDIATE ocorre APÓS LIBNAME mapping
- [ ] `checker.py` usa `WorkspaceClient.tables.get()` (UC REST API) — sem statement_execution
- [ ] Offline mode: sem credenciais → GateDecision="SKIP", não bloqueia
- [ ] `gate.py` implementa STRICT / LENIENT (threshold 20%) / FORCE / SKIP
- [ ] `reporter.py` gera `inventory_report.json` com summary + missing_tables + schemas_captured
- [ ] `reporter.save_schemas()` grava schemas.yaml no formato `list[dict]` (Story 5.1)
- [ ] CLI standalone: `sas2dbx inventory <dir>` funcional
- [ ] Testes unitários: extractor (patterns DATA/PROC SQL/MACRO), checker (mock UC), gate (4 modos), reporter
- [ ] `pytest` 100% verde

## Patterns de extração por tipo de bloco

```python
# DATA blocks
target: r"^\s*DATA\s+([\w.]+)\s*[;(]"
set:    r"\bSET\s+([\w.]+)"
merge:  r"\bMERGE\s+([\w.]+)"

# PROC blocks (FROM exige ponto para evitar subqueries)
from:   r"\bFROM\s+([\w]+\.[\w]+)"
join:   r"\bJOIN\s+([\w]+\.[\w]+)"
create: r"\bCREATE\s+TABLE\s+([\w.]+)"
data=:  r"\bDATA\s*=\s*([\w.]+)"
out=:   r"\bOUT\s*=\s*([\w.]+)"
```
